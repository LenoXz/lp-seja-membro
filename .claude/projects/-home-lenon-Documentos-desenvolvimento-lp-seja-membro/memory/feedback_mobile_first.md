name: Mobile-first design priority
description: Landing page is primarily for tablets and mobile devices, design mobile-first
type: feedback
---

A landing page é pensada primariamente para tablets e celulares. Cards devem ser grandes e o carrossel deve funcionar com swipe nativo.

**Why:** O público-alvo acessa via dispositivos móveis.
**How to apply:** Sempre priorizar o layout mobile, usar cards que ocupem ~85vw com snap-center, evitar componentes que dependam de hover.
